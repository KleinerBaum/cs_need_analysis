# Reports Archive

This directory is a historical archive for internal audits, deep-research notes,
Codex task lists, and generated local diagnostics. Files here are not the current
runtime contract for the app.

Current source-of-truth docs:

- `README.md` - current product flow, setup, verification, and CI contract.
- `AGENTS.md` - current coding-agent contract for this repository.
- `docs/legacy_wizard_modules.md` - archived wizard modules, replacement paths,
  route guardrails, and removal prerequisites.
- `docs/legacy_summary_outputs.md` - archived Summary output compatibility,
  including `employment_contract`.
- `docs/definition_of_done.md` - beta readiness and release-quality contract.

Historical reports may mention older product names, stale public GitHub state,
or superseded artifact scope. Treat those statements as point-in-time context
only. Before using any recommendation from this directory, validate it against
the current constants, route loader, tests, README, and AGENTS contract.

Generated reports from local or CI runs should stay under generated-output paths
such as `reports/junit/` or explicit smoke-test output files. Do not add new
tracked analysis reports without a clear owner, date, and current-status header.
